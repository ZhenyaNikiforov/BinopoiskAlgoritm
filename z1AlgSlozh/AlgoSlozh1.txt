int calc(int* arr, int size)
{
  int ans = 0;
  for (int i = 1; i < size; i++)
  {
    ans += arr[i] - arr[i-1];
  }
  return ans;
}

/*-Подпрограмма "calc" на языке C++ 
посвящена возвращению суммы разностей между текущим и предыдущим
элементом массива (ans = ans + (arr[i] - arr[i - 1]);)

Её алгоритмика времени по количеству элементов массива (n) составляет O(n), или y = f(x),
поскольку затраты времени на одну операцию ans = ans + (arr[i] - arr[i - 1]); линейны, и общее время
зависит только от количества элементов i.

Алгоритмика же по затратам временной памяти также O(n), поскольку затраты памяти на одну операцию также линейны.

Переменная же накопитель ans прирастает по памяти во время вполнения программы, но она не учитывается,
поскольку память, занимаемая ею, остаётся занятой и по завершении подпрограммы, когда подпрограмма возвращает результат,
то результат этот где-то хранится. -*/
